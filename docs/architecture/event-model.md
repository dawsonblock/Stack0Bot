# Event Model

Events are stored at `storage/runs/<runId>/events.jsonl`.

Important event types:
- `run_created`
- `intent_received`
- `intent_validated`
- `state_transition`
- `artifact_written`
- `command_executed`
- `model_called`
- `promotion_evaluated`
- `approval_recorded`
- `artifact_applied`
- `run_completed`
- `run_failed`
